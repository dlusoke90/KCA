Module 1 — Windows Tools

System & Administrative Tools

Access and understand the core Windows admin tools accessible via the Win+X Power User Menu.

📌
How to Open the Win+X MenuRight-click the Start button OR press Win + X on your keyboard. This opens the Power User Menu you saw in class.

📱 Apps and Features

📱
Settings

Apps and Features

View, modify, and uninstall all installed applications on your computer. Also lets you manage optional Windows features and app execution aliases.

🔋
Control

Mobility Center

Quick access to common laptop settings: display brightness, volume, battery status, wireless network, screen rotation, and presentation mode — all in one panel.

⚡
System

Power Options

Configure power plans (Balanced, Power Saver, High Performance). Control when the display turns off and when the PC goes to sleep. Critical for laptops and servers.

📋 Event Viewer

🔍 What is Event Viewer?

Event Viewer is a Windows administrative tool that lets you view detailed information about significant events on your computer — including system errors, security events, and application warnings.

Log TypeWhat It RecordsWhen to Use It
ApplicationEvents logged by applications/programsApp crashes, software errors
SecurityLogin attempts, file/object accessSecurity auditing, failed logins
SystemWindows system component eventsDriver failures, service stops
SetupApplication setup eventsInstallation issues

💻 System

💻

System Information

Shows your PC name, Windows edition, processor, RAM, system type (32/64-bit), and Windows activation status. The starting point for any tech support call.

🔧

Advanced System Settings

From here you access Performance settings, Environment Variables, Remote Desktop, and System Protection (restore points).

⌨️
Quick Access Shortcuts Open Apps & Features: Win+I → Apps
Open System: Win+Pause/Break or right-click This PC → Properties
Open Event Viewer: type eventvwr in Run dialog (Win+R)
Module 2 — Windows Tools

Device Manager & Network Connections

Manage hardware drivers and configure network adapters, VPNs, and connectivity settings.

🔌 Device Manager

What is Device Manager?

Device Manager is the central location for managing all hardware connected to your computer — from your keyboard to network cards to GPUs. It shows you what's installed, what's working, and what has problems.

Icon/StatusMeaningAction Needed
✅ No iconDevice working normallyNone
⚠️ Yellow triangleDevice has a problem or warningUpdate driver or check properties
❌ Red XDevice is disabledRight-click → Enable device
⬇️ Arrow downDevice manually disabledRight-click → Enable
❓ Question markUnknown device / no driverInstall driver from manufacturer
1

Update a Driver

Right-click the device → Update driver → Search automatically for drivers.

2

Roll Back a Driver

Right-click → Properties → Driver tab → Roll Back Driver (if a recent update caused problems).

3

Uninstall a Device

Right-click → Uninstall device. Useful when completely reinstalling a driver from scratch.

🌐 Network Connections

📶
Network

What You See Here

All network adapters on your PC: Ethernet, Wi-Fi, Bluetooth, VPN adapters, and virtual adapters. Each is shown with its current connection status.

🔧
Config

Key Tasks

Set a static IP address, change DNS servers, disable/enable a network adapter, bridge two connections, or diagnose a broken connection.

🔍
Diagnose

Adapter Properties

Right-click any adapter → Properties → double-click Internet Protocol Version 4 (TCP/IPv4) to set static IP, subnet mask, gateway, and DNS.

⚠️
ImportantDisabling your only active network adapter will immediately disconnect you from the internet and network. Always have a plan before making changes here.
Module 3 — Windows Tools

Disk Management & Computer Management

Manage storage volumes, partitions, and access all Windows admin tools from one unified console.

💾 Disk Management

What is Disk Management?

Disk Management lets you view and manage all physical disks and partitions on your system. You can create, delete, format, resize partitions and assign drive letters — without any third-party software.

TaskHow To Do ItUse Case
Create PartitionRight-click unallocated space → New Simple VolumeSetting up a new drive
Format VolumeRight-click partition → FormatPrepare drive for use (NTFS/FAT32)
Shrink VolumeRight-click partition → Shrink VolumeMake space for another partition
Extend VolumeRight-click partition → Extend VolumeAdd unallocated space to partition
Change Drive LetterRight-click → Change Drive Letter and PathsRename C:, D:, etc.
Mark as ActiveRight-click → Mark Partition as ActiveSet boot partition
⚠️
Danger ZoneDeleting or formatting a partition permanently destroys all data on it. Always verify you are working on the correct disk before proceeding.

🖥️ Computer Management

The All-in-One Admin Console

Computer Management bundles multiple administration tools into a single window. It's the most powerful single access point for Windows system administration.

📅

Task Scheduler

Create and manage automated tasks that run at specific times or when events occur — like backup scripts, cleanup jobs, or login scripts.

📋

Event Viewer

Same Event Viewer you learned earlier — accessible here too as part of the unified console.

📁

Shared Folders

See all shares, active sessions, and open files on the computer. Manage who has access to shared network folders.

👥

Local Users & Groups

Create and manage local user accounts and groups. Add users, reset passwords, disable accounts, and manage group membership.

📊

Performance Monitor

Track real-time CPU, memory, disk, and network performance with detailed graphs and reports.

🔧

Device Manager

The same Device Manager from the Win+X menu — included here in the unified console.

💾

Disk Management

The storage management tool — also accessible from within Computer Management.

⚙️

Services

Start, stop, pause, or restart Windows services. Configure startup type (Automatic, Manual, Disabled). Critical for IT support.

Module 4 — Windows Tools

Windows PowerShell & Run Dialog

Automate Windows with PowerShell scripts and quickly open any tool using the Run command.

🖥️ Windows PowerShell vs PowerShell (Admin)

🟦

Windows PowerShell

Runs with your normal user permissions. Good for most scripting, querying system info, and file operations that don't require elevated access.

🔴

PowerShell (Admin)

Runs with Administrator privileges. Required for system changes: installing software, modifying registry, managing services, and network configuration changes.

⚠️
Always Use Admin Mode CarefullyCommands run in PowerShell (Admin) have full system access. A typo can delete system files or break Windows. Double-check every command before pressing Enter.

⚡ Essential PowerShell Commands

Get-ProcessList all running processesGet-Process | Where CPU -gt 10

Module 5 — Windows Tools

Task Manager, File Explorer, Search & More

Monitor processes, navigate your files, and control your Windows desktop environment.

📊 Task Manager

Your Real-Time PC Monitor

Task Manager shows you everything running on your PC right now — processes, CPU usage, RAM, disk, network activity. It's the first tool to open when your computer is slow.

TabWhat It ShowsKey Use
ProcessesAll running apps and background tasks with resource usageKill frozen apps, see what's eating CPU
PerformanceReal-time graphs: CPU, Memory, Disk, Network, GPUDiagnose slowdowns, see live utilization
App historyHistorical resource usage per appFind resource-hungry applications
StartupApps that launch at Windows startupDisable slow-boot programs
UsersAll logged-in users and their resource usageMulti-user session management
DetailsAdvanced view: PID, status, username per processIT troubleshooting, malware hunting
ServicesAll Windows services with statusQuick service start/stop/restart
💡
Task Manager Shortcuts Open Task Manager: Ctrl+Shift+Esc (fastest method)
Or: Ctrl+Alt+Del → Task Manager
End a frozen program: select it → End Task

📁 File Explorer

📁

Navigation Pane

Left panel shows Quick Access, OneDrive, This PC (drives), and Network. Pin frequently used folders to Quick Access for fast navigation.

👁️

View Options

Switch between Large Icons, List, Details, and Tiles views. Enable "Hidden Items" and "File Name Extensions" in View → Show.

🔎

Search in Explorer

Use the search box (top right) to search within a folder. Use filters like date:yesterday, size:large, or type:.pdf.

The Remaining Win+X Items

🔍
Search

Search

Opens Windows Search. Find apps, files, settings, and web results. Press Win+S to open instantly. Supports natural language queries.

⚙️
System

Settings

The modern Windows Settings app — replaces much of the old Control Panel. Configure accounts, network, display, privacy, updates, and more. Open with Win+I.

🖥️
Desktop

Desktop

Minimizes all open windows and shows your desktop. Same as pressing Win+D.

🔴
Power

Shut Down or Sign Out

Sub-menu options: Sign out, Sleep, Hibernate, Shut down, Restart. Shut down closes all programs and fully powers off. Restart is the fix for most Windows issues.

Module 6 — Microsoft Office

Microsoft Word — Document Creation

Create professional documents, format text, use styles, and collaborate with others.

🎯 The Word Interface

AreaWhat It Does
Quick Access ToolbarTop-left icons for Save, Undo, Redo — customizable
RibbonOrganized tabs (Home, Insert, Design, Layout, etc.) with all features
Document AreaThe main white area where you type and see your document
Status BarBottom bar showing page number, word count, zoom level, view mode
RulersTop and left rulers for measuring margins and setting tab stops

📑 Ribbon Tabs Overview

🏠

Home Tab

Font (Bold, Italic, Underline, size, color), Paragraph (alignment, bullets, spacing), Styles (Heading 1, Normal, etc.), Clipboard (Cut, Copy, Paste).

➕

Insert Tab

Add Tables, Pictures, Icons, Shapes, SmartArt, Charts, Headers & Footers, Page Numbers, Hyperlinks, Comments, and Equations.

🎨

Design Tab

Document Themes (color + font combinations) and Watermarks. Change the entire look of a document with one click.

📐

Layout Tab

Margins, Orientation (Portrait/Landscape), Paper Size, Columns, Line Breaks, Spacing, and Indentation.

📚

References Tab

Table of Contents, Footnotes, Citations, Bibliography, Captions, Index — for professional academic and business documents.

✉️

Review Tab

Spelling & Grammar check, Thesaurus, Track Changes, Comments, Compare documents, Protect document with a password.

⭐ Key Word Features

Styles — The Most Important Feature

Styles (Heading 1, Heading 2, Normal, etc.) let you format your document consistently and automatically generate a Table of Contents. Always use Styles instead of manually bolding/resizing text.

Track Changes

Review tab → Track Changes. Every edit is recorded with the author's name and timestamp. The recipient can Accept or Reject each change. Essential for collaborative work.

Find & Replace

Ctrl+H opens Find & Replace. Instantly swap every instance of a word in the entire document. Supports wildcards and formatting replacement.

Page Layout & Margins

Layout tab → Margins. Standard is 1 inch all around. Use Narrow (0.5") for more content per page. Orientation: Portrait = taller than wide, Landscape = wider than tall.

Module 7 — Microsoft Office

Microsoft Excel — Spreadsheets & Data

Organize data, perform calculations with formulas, and visualize information with charts.

🧱 Excel Building Blocks

🔲

Cells

The smallest unit. Every cell has an address: column letter + row number (e.g., A1, C5). Click to select, type to enter data.

↕️

Rows & Columns

Rows are numbered (1, 2, 3...). Columns are lettered (A, B, C... Z, AA, AB...). A typical spreadsheet has 1,048,576 rows and 16,384 columns.

📋

Worksheets

Tabs at the bottom of the window. One Excel file (workbook) can contain multiple worksheets. Right-click a tab to rename, move, copy, or delete it.

🧮 Essential Formulas

📌
All formulas start with = (equals sign)For example: =SUM(A1:A10) or =A1+B1. If you don't start with =, Excel treats it as plain text.
FormulaWhat It DoesExample
=SUM()Adds all numbers in a range=SUM(A1:A10)
=AVERAGE()Calculates the mean of a range=AVERAGE(B2:B20)
=COUNT()Counts cells with numbers=COUNT(C1:C100)
=MAX()Returns the largest value=MAX(D1:D50)
=MIN()Returns the smallest value=MIN(D1:D50)
=IF()Returns one value if true, another if false=IF(A1>100,"Pass","Fail")
=VLOOKUP()Looks up a value in a table=VLOOKUP(A1,B:C,2,FALSE)
=CONCATENATE()Joins text from multiple cells=CONCATENATE(A1," ",B1)
=TODAY()Returns today's date=TODAY()
=ROUND()Rounds a number to specified decimals=ROUND(A1,2)

📊 Creating Charts

1

Select Your Data

Highlight the cells with the data you want to visualize, including headers.

2

Insert → Charts

Go to Insert tab → choose chart type: Column, Bar, Line, Pie, Scatter, etc.

3

Customize the Chart

Click the chart → use the Chart Design and Format tabs to add titles, change colors, and adjust the legend.

4

Move or Resize

Drag the chart to reposition it. Drag the corners to resize. You can also move it to its own sheet.

🔑 Key Excel Features

🔽

AutoFilter

Data tab → Filter. Adds dropdown arrows to column headers so you can filter rows by any criteria — dates, text, numbers, custom conditions.

🎨

Conditional Formatting

Home tab → Conditional Formatting. Automatically color cells based on their value — red for negative, green for positive, data bars, heat maps.

📌

Freeze Panes

View tab → Freeze Panes. Keeps header rows/columns visible as you scroll through large datasets. Essential for big spreadsheets.

🔒

Protect Sheet

Review tab → Protect Sheet. Lock cells so others can't accidentally change formulas. Set a password to prevent unprotecting.

Module 8 — Microsoft Office

Microsoft PowerPoint — Presentations

Create compelling slide decks with layouts, transitions, animations, and media.

🖼️ The PowerPoint Interface

AreaPurpose
Slide Panel (left)Thumbnail view of all slides — drag to reorder, right-click to duplicate/delete
Editing Area (center)The main slide — click any element to edit it
Notes Panel (bottom)Speaker notes for each slide — not visible to audience, shown to presenter
RibbonHome, Insert, Design, Transitions, Animations, Slide Show, Review, View tabs

🎨 Key PowerPoint Tabs

🎨

Design Tab

Choose a Theme (colors + fonts + backgrounds for the whole deck), Slide Size (Standard 4:3 or Widescreen 16:9), and Format Background.

✨

Transitions Tab

Visual effects that play when you move from one slide to the next — Fade, Push, Wipe, Morph, etc. Apply to one slide or all slides at once.

🎬

Animations Tab

Motion effects on individual elements within a slide — Appear, Fly In, Fade, Spin, etc. Set to trigger On Click, After Previous, or With Previous.

▶️

Slide Show Tab

Start the presentation from the beginning or current slide. Set up Presenter View (see notes + next slide on your screen, audience sees only the slide).

💡 Slide Design Best Practices

✅
DO: One main idea per slide. Use images over text. Keep text large (28pt+ for body). Use consistent colors from your theme. Include slide numbers.
❌
AVOID: Walls of text. More than 6 bullet points per slide. Too many animations. More than 3 font sizes. Reading your slides word-for-word.

🔧 Essential PowerPoint Techniques

1

Slide Master

View → Slide Master. Edit the master template to apply consistent fonts, colors, and logos to every slide automatically.

2

Align & Distribute

Select multiple objects → Home → Arrange → Align. Makes elements perfectly centered or evenly spaced without guessing.

3

Group Objects

Select multiple shapes/images → right-click → Group. Move and resize them together as one unit.

4

Export to PDF

File → Export → Create PDF/XPS. Preserves formatting perfectly. Recipients don't need PowerPoint to view it.

5

Presenter View

During your presentation, press Alt+F5 to use Presenter View — see your notes and the next slide while the audience sees only the current slide.
