Common Threats

Phishing: fake emails. DoS/DDoS: overwhelm resources. Man-in-the-Middle: intercept traffic. SQL Injection: exploit databases. Ransomware: encrypt data for ransom. Social Engineering: manipulate people.

Security Best Practices

AAA: Authentication, Authorization, Accounting. Principle of Least Privilege. Defense in Depth (multiple layers). Patch management. Disable unused services. Strong passwords + MFA. Segment networks.

AAA frameworkLeast privilegeDefense in depthMFA required

Cisco Device Security

Banner motd: legal warning. service password-encryption: encrypt passwords. enable secret: encrypted enable password. SSH: use instead of Telnet. Login local: local user database. Exec-timeout: auto logout.

banner motdservice password-encryptionenable secretSSH not Telnet
📡

Port Security

Limit MAC addresses per port. Modes: protect (drop silently), restrict (drop + log), shutdown (err-disable). Sticky: dynamically learn and save MACs. Maximum MACs: 1 by default. Prevents MAC flooding.

Limit MACs per portProtect/Restrict/ShutdownSticky learningDefault max = 1
