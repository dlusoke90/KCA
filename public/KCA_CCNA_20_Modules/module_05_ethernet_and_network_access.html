Core Concepts

Ethernet Standards

10BASE-T (10Mbps), 100BASE-TX Fast Ethernet, 1000BASE-T Gigabit (Cat5e), 10GBASE-T (Cat6a). Fiber: 1000BASE-SX (multimode), 1000BASE-LX (singlemode). IEEE 802.3 standard.

10Mbps: 10BASE-T100Mbps: FastEthernet1Gbps: GigabitEthernet10Gbps: 10GBASE-T
📦

Ethernet Frame

Preamble(7B) + SFD(1B) + Dst MAC(6B) + Src MAC(6B) + Type/Length(2B) + Data(46-1500B) + FCS(4B). Min frame 64B, max 1518B. MTU typically 1500 bytes.

Preamble 7BMAC addresses 6B eachData 46-1500BFCS 4B CRC
🏷️

MAC Addresses

48-bit (6 octets) hardware address burned into NIC. Format: 00:50:79:66:68:00. First 24 bits = OUI (vendor). Last 24 bits = device ID. Switches build MAC address tables.

48-bit hardwareOUI: first 24 bitsBIA: burned-inSwitches use MACs
🔄

CSMA/CD vs CSMA/CA

CSMA/CD: wired Ethernet - detect collisions (legacy hubs). CSMA/CA: wireless 802.11 - avoid collisions. Full-duplex switches eliminated collisions. Half-duplex uses CSMA/CD.

CSMA/CD: wiredCSMA/CA: wirelessFull-duplex: no collisionHalf-duplex legacy
