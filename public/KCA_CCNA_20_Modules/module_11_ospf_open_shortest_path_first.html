OSPF Overview

Link-state routing protocol. RFC 2328. Uses Dijkstra's SPF algorithm. Administrative Distance = 110. Hello packets every 10s (30s NBMA). Dead interval = 4x Hello. Classless, supports VLSM.

Link-state protocolAD = 110SPF algorithmHello 10s, Dead 40s
👑

DR/BDR Election

On multi-access networks (Ethernet). Highest priority wins (default 1, range 0-255). Tie: highest Router ID. Router ID: highest loopback IP or highest active interface IP. Priority 0 = never DR.

Highest priority = DRTie: highest Router IDPriority 0 = never DRBDR = backup DR
🌍

OSPF Areas

Area 0 = backbone (required). All areas must connect to Area 0. ABR = Area Border Router (connects areas). ASBR = redistributes external routes. Stub areas reduce LSA flooding.

Area 0 backboneABR connects areasASBR external routesStub reduces LSAs
⚙️

OSPF Configuration

router ospf 1 → router-id 1.1.1.1 → network 192.168.1.0 0.0.0.255 area 0. Wildcard mask = inverse of subnet mask. Check neighbors: show ip ospf neighbor. Verify routes: show ip route ospf.

router ospf 1router-idnetwork + wildcardarea 0 backbone
