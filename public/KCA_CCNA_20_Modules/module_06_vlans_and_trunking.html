Core Concepts

What are VLANs?

Virtual LANs segment network at Layer 2 without physical separation. Benefits: security, broadcast control, flexibility. Default VLAN 1. Native VLAN (untagged). VLANs 1-4094.

Segment L2 trafficReduce broadcastsImprove securityVLANs 1-4094
🔗

802.1Q Trunking

Trunk carries multiple VLANs between switches. 802.1Q adds 4-byte tag to frame. Native VLAN sent untagged. DTP (Dynamic Trunking Protocol) negotiates trunks automatically.

802.1Q 4-byte tagNative VLAN untaggedDTP auto-negotiateTrunk = multi-VLAN
⚙️

VLAN Configuration

vlan 10 → name SALES. interface gi0/1 → switchport mode access → switchport access vlan 10. Trunk: switchport mode trunk → switchport trunk allowed vlan 10,20,30.

vlan 10switchport mode accessswitchport access vlanswitchport mode trunk
🌿

VTP (VLAN Trunking Protocol)

Cisco proprietary. Propagates VLAN config across switches. Modes: Server (can create/modify), Client (receives), Transparent (forwards only). VTP domain must match.

Server: creates VLANsClient: receivesTransparent: forwardsDomain must match
