Core Concepts
📡

TCP/IP Model Layers

4 layers: Network Access (L1+L2), Internet (L3), Transport (L4), Application (L5-7). Maps to OSI but simplified. Used in practice for all internet communications.

Network AccessInternetTransportApplication
🔄

TCP vs UDP

TCP: reliable, ordered, connection-oriented, 3-way handshake (SYN→SYN-ACK→ACK). UDP: fast, unreliable, connectionless. TCP=HTTP/FTP/SSH. UDP=DNS/DHCP/VoIP/streaming.

TCP: ReliableUDP: FastTCP: SYN/ACKUDP: No handshake
🌍

Core Protocols

HTTP/HTTPS (80/443), SSH (22), Telnet (23), FTP (21/20), SMTP (25), DNS (53), DHCP (67/68), SNMP (161), BGP (179), OSPF (89).

HTTP 80HTTPS 443SSH 22DNS 53DHCP 67/68
🤝

TCP Three-Way Handshake

Step 1: Client sends SYN. Step 2: Server replies SYN-ACK. Step 3: Client sends ACK. Connection established! Used for all TCP sessions before data transfer.

1. SYN →2. ← SYN-ACK3. ACK →Connected! ✓
