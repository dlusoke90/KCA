DNS, NTP & IP Services

📚 4 Core Topics 💻 5 CLI Commands ❓ 5 Quiz Questions
Core Concepts
🔍

DNS (Domain Name System)

Translates hostnames to IPs. Hierarchical: Root → TLD (.com) → Domain (cisco.com) → Host (www). Records: A (IPv4), AAAA (IPv6), MX (mail), CNAME (alias), PTR (reverse). Port 53 UDP/TCP.

A record: IPv4AAAA: IPv6MX: mailCNAME: aliasPTR: reverse
⏰

NTP (Network Time Protocol)

Synchronizes clocks across network. Stratum 0 = atomic clock. Stratum 1 = directly connected to stratum 0. Stratum 2 = synced to stratum 1. Port 123 UDP. Critical for logs, certificates, Kerberos.

Stratum 0: atomicStratum 1: primaryStratum 2: secondaryPort 123 UDP
📊

SNMP (Simple Network Management Protocol)

Monitors and manages network devices. Manager polls agents. Agent sends traps for alerts. MIB = database of manageable objects. SNMPv3 = encrypted, authenticated (use this!).

Manager polls agentsTraps = alertsMIB databaseSNMPv3 = secure
🌐

Other IP Services

Syslog: centralized logging (port 514 UDP). TFTP (port 69): transfer IOS images. FTP (21/20): file transfer. SCP: secure file copy. NetFlow: traffic analysis. CDP/LLDP: neighbor discovery.

Syslog: port 514TFTP: port 69FTP: 21/20CDP/LLDP: discovery
