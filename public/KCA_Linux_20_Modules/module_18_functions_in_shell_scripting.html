Functions in Shell Scripting

🐧 4 Topics 💻 5 CLI Commands ❓ 5 Quiz Questions
Core Concepts

Defining Functions

greet() { echo "Hello, $1!" } greet 'KCA' $1, $2 = positional parameters Define before calling!

Parameters & Return

$1 $2 ... : Positional params $# : Number of arguments $@ : All arguments return 0 : Success return 1 : Failure

Local Variables

calculate() { local RESULT=$(( $1 + $2 )) echo $RESULT } SUM=$(calculate 5 3) 'local' prevents variable leaking

Practical Examples

check_root() { [ $(id -u) -ne 0 ] && echo 'Run as root' && exit 1; } log() { echo "[$(date)] $1" >> script.log; } backup() { cp "$1" "$1.bak" && echo "Backed up $1"; }

Live CLI Commands — Click ▶ Run on each command
TERMINAL SIMULATOR
$ greet() { echo "Hello, $1! Welcome to KCA."; } && greet 'Student'
$ add() { local R=$(( $1 + $2 )); echo $R; } && SUM=$(add 10 5) && echo "10 + 5 = $SUM"
$ is_even() { [ $(( $1 % 2 )) -eq 0 ] && echo 'Even' || echo 'Odd'; } && is_even 4 && is_even 7
$ repeat() { for i in $(seq 1 $2); do echo $1; done; } && repeat 'KCA' 3
$ info() { echo "Args: $#"; echo "All: $@"; } && info one two three
