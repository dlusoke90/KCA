Loops in Shell Scripting

🐧 4 Topics 💻 5 CLI Commands ❓ 5 Quiz Questions
Core Concepts

for Loop

for i in 1 2 3 4 5; do echo "Number: $i" done for file in *.txt; do echo "Processing $file" done

while Loop

COUNT=1 while [ $COUNT -le 5 ]; do echo "Count: $COUNT" COUNT=$((COUNT + 1)) done

until Loop

until [ $COUNT -gt 5 ]; do echo $COUNT COUNT=$((COUNT+1)) done (Runs until condition is TRUE)

Loop Control

break : Exit the loop continue : Skip to next iteration break 2 : Break 2 nested loops
