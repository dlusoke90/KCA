Basic File & Directory Commands

🐧 4 Topics 💻 5 CLI Commands ❓ 5 Quiz Questions
Core Concepts

Creating Files & Dirs

touch file.txt : Create empty file mkdir mydir : Create directory mkdir -p a/b/c : Create nested dirs echo 'text' > file.txt : Create with content

Copying & Moving

cp file.txt backup.txt : Copy file cp -r dir1 dir2 : Copy directory mv file.txt /tmp/ : Move file mv old.txt new.txt : Rename file

Deleting Files

rm file.txt : Delete file rm -r mydir : Delete directory rm -rf mydir : Force delete ⚠️ No recycle bin in Linux!

Viewing Files

cat file.txt : Display file less file.txt : Page through head -10 file.txt : First 10 lines tail -10 file.txt : Last 10 lines wc -l file.txt : Count lines
