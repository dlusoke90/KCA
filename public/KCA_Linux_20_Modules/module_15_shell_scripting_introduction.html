Core Concepts

What is a Shell Script?

A text file with shell commands that automates tasks. Use .sh extension. First line shebang: #!/bin/bash Tells OS which interpreter to use.

Your First Script

#!/bin/bash echo 'Hello KCA!' date whoami Save as hello.sh chmod +x hello.sh Run: ./hello.sh

Variables

NAME='KCA' echo $NAME echo ${NAME} read -p 'Enter name: ' NAME No spaces around = sign!

Best Practices

Start with #!/bin/bash Add comments with # set -e : Exit on error set -x : Debug mode Quote variables: "$VAR"

$ NAME='KCA Student' && echo "Hello, $NAME!"
$ echo "Today is: $(date +%A)"
$ echo "Files in /etc: $(ls /etc | wc -l)"
