Core Concepts

if/else Syntax

if [ condition ]; then commands elif [ condition ]; then commands else commands fi

Comparison Operators

Numbers: -eq -ne -lt -le -gt -ge Strings: = != -z (empty) -n (not empty) Files: -f (file) -d (dir) -e (exists) -r (readable) -w (writable) -x (executable)

case Statement

case $VAR in pattern1) commands;; pattern2) commands;; *) default;; esac

Exit Codes

0 = success, non-zero = failure $? holds last exit code exit 0 : Success exit 1 : Error
