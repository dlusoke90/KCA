Text Editors — nano & vim

🐧 4 Topics 💻 5 CLI Commands ❓ 5 Quiz Questions
Core Concepts

nano Editor

Open: nano file.txt Save: Ctrl+O → Enter Exit: Ctrl+X Search: Ctrl+W Cut line: Ctrl+K Paste: Ctrl+U

vim Editor

Modes: Normal (navigate), Insert (type), Visual (select) Open: vim file.txt Insert mode: i Save & quit: :wq Quit no save: :q!

vim Navigation

h=left j=down k=up l=right gg=go to start G=go to end dd=delete line yy=copy line p=paste /word=search

Choosing an Editor

nano: quick edits, beginners vim: power users, always available gedit: GUI editor VS Code: modern IDE
