Disk & Storage Management

🐧 4 Topics 💻 5 CLI Commands ❓ 5 Quiz Questions
Core Concepts

Disk Usage

df -h : Disk space (human readable) du -sh /var : Size of directory du -sh * : Size of all items lsblk : List block devices

Mount & Unmount

mount /dev/sdb1 /mnt : Mount disk umount /mnt : Unmount /etc/fstab : Permanent mounts on boot

File System Types

ext4 : Most common Linux FS xfs : High-performance (RHEL) btrfs : Modern with snapshots ntfs : Windows FS FAT32 : Cross-platform USB

Disk Tools

mkfs.ext4 /dev/sdb1 : Format fsck /dev/sdb1 : Check/repair parted : Partition management lvs/pvs/vgs : LVM management
