Core Concepts

Linux Directory Structure

Linux uses a hierarchical file system starting from / (root). Everything — files, devices, processes — is a file under /.

Key Directories

/ : Root directory /home : User home directories /etc : Configuration files /var : Variable data (logs) /bin : Essential binaries /usr : User programs /tmp : Temporary files /dev : Device files

Absolute vs Relative Paths

Absolute: starts from / → /home/user/file.txt Relative: from current dir → ./file.txt ~ represents your home directory

Navigating

cd /etc : Go to /etc cd ~ : Go home cd .. : Go up one level cd - : Previous directory
