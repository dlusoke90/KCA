User & Group Management

🐧 4 Topics 💻 5 CLI Commands ❓ 5 Quiz Questions
Core Concepts

User Accounts

Linux is multi-user. /etc/passwd : User info /etc/shadow : Encrypted passwords Each user has: username, UID, GID, home dir, shell

Managing Users

useradd -m -s /bin/bash username : Add user passwd username : Set password usermod -aG sudo username : Add to sudo userdel -r username : Delete user

Groups

groupadd mygroup : Create group groupdel mygroup : Delete group groups username : Show groups id username : Show UID, GID, groups

sudo & Root

sudo command : Run as root su - username : Switch user sudo -i : Open root shell /etc/sudoers : Controls sudo access
